Capture lost crossing into the DevTools pane

Top-level page, no author iframe. Sibling of issue 417662678 (iframe boundary), with the docked DevTools render surface as the boundary instead.

  1. Dock DevTools to the right (or bottom).
  2. Press the box (turns green) and hold the button.
  3. Drag the cursor into the DevTools pane and release the button there.
  4. Repeat several times — intermittent, reproduces roughly 1 in 10 drags.

Pass (no bug): the box turns gray — its own pointerup ran. Fail (bug): the box stays green — lostpointercapture fired before the release, the pointerup retargeted to HTML, and the listener on the box never saw it. Only the document listener did.
